## Variable Declaration
Different ways to declare and initialize variables
```python
name = "John"           # String
age = 30               # Integer
height = 5.9           # Float
is_active = True       # Boolean
fruits = ["apple", "banana"]  # List
x, y = 10, 20          # Multiple assignment

print(f"Name: {name}, Age: {age}, Height: {height}")
```
