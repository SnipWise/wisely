## Data Types
Working with basic data types
```python
# Numbers
integer = 42
float_num = 3.14
complex_num = 1 + 2j

# Strings
string = "Python"
multiline = """This is a
multiline string"""

# Collections
list_example = [1, 2, 3, 4]
tuple_example = (1, 2, 3)
set_example = {1, 2, 3}
dict_example = {"name": "Alice", "age": 30}

print(f"Types: {type(integer)}, {type(string)}, {type(list_example)}")
```
